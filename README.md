## 🌟 Engineering Principles I Live By

<div align="center">

| Principle | Approach |
|-----------|----------|
| 🧱 **Design First** | Schema, contracts & API design before writing code |
| ⚡ **Performance** | Cache aggressively, index smartly, query efficiently |
| 🔐 **Security** | Auth at every layer — JWT, rate limiting, input validation |
| 📦 **Modularity** | Clean separation of concerns, SOLID principles always |
| 🔭 **Observability** | Every service ships with metrics, logs & traces |
| 🔁 **Resilience** | Circuit breakers, retries, dead letter queues |

</div>
